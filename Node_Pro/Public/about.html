NodeJS

What is Node.JS? Node.JS is javascript runtime(not a language or framework) Node.JS is built on V8 Javascript engine. which is written on C++. Why we Use Node?. Node is fast,efficient and highly scalable. Node is Event driven, non-blocking I/O Model A Node.js app runs in a single process, without creating a new thread for every request. Same language on the both frontend and backend(JS). Works on Single thread using non-blocking I/O calls. It supports tens of thousands concurrent connections. EventEmitter class is used to bind events and listners NPM-Node Package Manager it is used to instal 3rd party packages(frameworks,librarires). Packages get stored in "node_modules" folder. All depencies are listed in "package.json" file. NPM scripts can be created to run certain tasks. If node installed we can run javascript from console itself using command Node.
